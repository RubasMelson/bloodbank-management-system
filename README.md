# How to Run the Blood Bank Project

Since you are using XAMPP, follow these simple steps to run your application.

## 1. Start the Server
1. Open **XAMPP Control Panel**.
2. Click **Start** for **Apache** (Web Server).
3. Click **Start** for **MySQL** (Database).

## 2. Access the Application
Open your web browser (Chrome, Edge, etc.) and go to this URL:

**[http://localhost/bloodbank](http://localhost/bloodbank)**

## 3. Login Credentials (Default)
If you haven't changed them, these are likely the credentials from your database setup:

- **Admin Login**:
  - Email: `admin@bloodbank.com`
  - Password: `password123` (or check your `bloodbank.sql` file if unsure)

## Troubleshooting
- **Database Error?** Ensure your database is named `bloodbank` in phpMyAdmin (`http://localhost/phpmyadmin`).
- **404 Not Found?** Ensure your project folder is exactly at `c:\xampp\htdocs\bloodbank`.
